# Portfolio Website

A modern, responsive personal portfolio website showcasing skills, projects, and contact information.

## Features

- **Responsive Design**: Fully responsive layout that works on all devices (desktop, tablet, mobile)
- **Modern UI/UX**: Clean and professional design with smooth animations
- **Interactive Navigation**: Fixed navigation bar with smooth scrolling
- **Sections**:
  - Hero section with introduction
  - About section with personal information
  - Skills showcase with technology stack
  - Featured projects gallery
  - Contact information with social media links

## Technologies Used

- HTML5
- CSS3 (with CSS Variables and Flexbox/Grid)
- JavaScript (Vanilla JS)
- Font Awesome Icons

## Getting Started

1. Clone the repository
2. Open `index.html` in your web browser
3. Customize the content with your own information

## Customization

To personalize this portfolio:

1. **Update Personal Information**: Edit `index.html` to replace placeholder text with your details
2. **Add Projects**: Modify the projects section with your actual projects
3. **Update Contact Info**: Change email, phone, and social media links
4. **Customize Colors**: Modify CSS variables in `styles.css` to match your brand
5. **Add Images**: Replace project placeholders with actual project screenshots

## Live Demo

Open `index.html` in any modern web browser to view the portfolio.
